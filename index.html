Scout Janssen

Scout Janssen is a Robot Scouting Tool built for the FIRST Robotics Competition (FRC).

Karten

Karten is a tool built in React to quickly create flashcards for foreign languages.

Karten supports any languge supplied by the Yandex API, and is able to convert to Quizlet Study Sets.
